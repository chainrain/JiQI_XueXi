机器学习的概念是:
从数据中自动分析获得某种规律,然后利用规律对未来进行预测
也能让程序替换人手的步骤,减少企业的成本同时提高效率

比如说:
信贷评估
信用评估
自然语言处理
图像识别等

为什么需要机器学习:
1.解放生产力,例如:智能客服.
优点:机器能24小时运作,人不行.
2.解决专业复杂问题,例如:ET医疗,能辅助医生判断病人的情况,比如说脑电图,心电图领域
优点:提高效率
3.行为分析,例如:电商首页广告,可利用客人的搜索习惯,推荐合适的商品给用户

需要的领域:
医疗,航空,教育,物流,电商等

需要准备的:
读取工具:pandas
数组工具:numpy
特征处理:sklearn

机器学习比较重要的一块:特征工程
特征工程是将原始数据转换为更好地代表预测模型的潜在问题的特征的过程,
从而提高了对未知数据的预测准确性
用到的工具:Scikit-learn
Scikit-learn库,是python语言的机器学习工具
Scikit-learn包括许多机器学习算法
Scikit-learn文档完善
Scikit-learn安装:pip install Scikit-learn
注意:安装Scikit-learn工具,要安装Numpy和pandas等的数据分析库,所以在这我是用了之前数据分析的虚拟环境
